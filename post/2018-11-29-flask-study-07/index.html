Flask Study 07 jinja模板基础使用

2018-11-29 Liseen

人生苦短,我学Python… Flask默认的模板引擎是Jinja2类似php内的smarty甚是好用

文件目录结构如下

├── app.py  # 入口
├── static  # 静态资源
│   ├── avatar.jpg
│   ├── css
│   ├── favicon.ico
│   ├── js
│   └── style.css
└── templates   # 模板文件夹
    ├── base.html
    ├── errors
    ├── index.html
    ├── macros.html
    └── watchlist.html

app.py内定义了视图函数对应的渲染逻辑

from flask import Flask, render_template,flash,redirect,url_for
app = Flask(__name__)
app.secret_key = 'dev'

user = {
    'username': 'Grey Li',
    'bio': 'A boy who loves movies and music.',
}

movies = [
    {'name': 'My Neighbor Totoro', 'year': '1988'},
    {'name': 'Three Colours trilogy', 'year': '1993'},
    {'name': 'Forrest Gump', 'year': '1994'},
    {'name': 'Perfect Blue', 'year': '1997'},
    {'name': 'The Matrix', 'year': '1999'},
    {'name': 'Memento', 'year': '2000'},
    {'name': 'The Bucket list', 'year': '2007'},
    {'name': 'Black Swan', 'year': '2010'},
    {'name': 'Gone Girl', 'year': '2014'},
    {'name': 'CoCo', 'year': '2017'},
]

@app.route('/watchlist')
def watchlist():
    return render_template('watchlist.html',user=user,movies=movies)

@app.route('/')
def index():
    return render_template('index.html')

@app.context_processor
def inject_info():
    foo = 'I am foo.'
    return dict(foo=foo)

@app.route('/flash')
def just_flash():
    flash('你好,我是闪电!')
    return redirect(url_for('index'))

@app.errorhandler(404)
def page_not_found(e):
    return render_template('errors/404.html'), 404

带集成的模板引擎自然少不了复用功能 base.html

<!DOCTYPE html>
<html lang="en">
<head>
    {% block head %}
     <meta charset="UTF-8">
     <title>{% block title %}Template - HelloFlask {% endblock %}</title>
    {% endblock %}
    {% block styles %}
        <link rel="stylesheet" type="text/css" href="{{ url_for('static',filename='style.css') }}">
    {% endblock %}
</head>
<body>
<na>
    <ul>
        <li>
            <a href="{{ url_for('index') }}">Home</a>
        </li>
    </ul>
</na>
<main>
    {% for message in get_flashed_messages() %}
        <div class="alert">{{ message }}</div>
    {% endfor %}

    {% block content %}{% endblock %}
</main>
<footer>
    {% block footer %}
        <div>This a Footer</div>
    {% endblock %}
</footer>
{% block scripts %}

{% endblock %}
</body>
</html>

index.html 集成了base.html

{% extends 'base.html' %}
{% from 'macros.html' import qux %}
{% block content %}
    {% set iterated = true %}
    <h1>Template</h1>
    <ul>
    <li>
        <a href="{{ url_for('watchlist') }}">WatchList</a>
    </li>
    <li>{{ foo }}</li>
    <li>Macro: {{ qux(amount=2) }}</li>
    {% if iterated %}
        <li>iterated is True</li>
    {% endif %}
    </ul>
{% endblock %}

watchlist.html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{{ user.username }} 's watchList</title>
</head>
<body>
<a href="{{ url_for('index')}}"> &larr; Return</a>
<h2>{{user.username}}</h2>
{% if user.bio %}
<i>{{user.bio}}</i>
{% else %}
<i>This user has not provided a bio.</i>
{% endif %}
{# Below is the movie list (this is comment) #}
<h5>{{user.username}} 's WatchList ({{movies|length}}):</h5>
<ul>
    {% for movieitem in movies %}
    <li>{{movieitem.name}}-{{movieitem.year}}</li>
    {% endfor %}
</ul>
</body>
</html>

有些情况下我们需要复用一些常用的模块比如说弹窗啊什么的,这时候就用到了宏这个特性.

{% macro qux(amount=1)%}
   {% if amount == 1 %}
       I am qux
   {% elif amount > 1 %}
       We are quxs
   {% endif %}
{% endmacro %}

调用方式如下:

{% from 'macros.html' import qux %}
<li>Macro: {{ qux(amount=2) }}</li>
